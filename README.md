响应式购物页面
一个基于 HTML5 / CSS3 / JavaScript 开发的简易电商购物页面，实现了基础购物流程与多设备适配。

功能介绍
商品展示：展示商品图片、名称、价格
购物车交互：支持商品添加/删除、数量增减、总价实时计算
响应式布局：自动适配手机、平板、电脑等不同设备屏幕

如何运行
1. 将商品图片文件（手机.jpg、电脑.jpg、耳机.jpg等）与 `index.html` 放在同一个文件夹
2. 直接打开 `index.html` 文件，即可在浏览器中预览项目
3. 也可通过 GitHub Pages 在线访问：https://liu9-66.github.io/~/

技术栈
HTML5：页面结构搭建
CSS3：响应式布局与样式美化
JavaScript：交互逻辑实现（购物车计算、事件处理等）
Git：版本控制
